# Responsible Use

## Decision-Support Scope

Outputs are analyst decision-support signals. The system provides confidence-aware root-cause-related factor indicators from free-text incident narratives.

## No Causal Proof

The model does not prove causality and does not produce definitive root cause determinations. Predictions should be interpreted as contributory-factor indicators requiring context review.

## Human Review Requirement

Human-in-the-loop review is required for any operational use. Analysts should evaluate:
- prediction confidence and review flags
- local context and operational records
- investigation findings outside the model

## Operational Safety Boundary

This prototype is not certified for operational safety decisions and should not be used as an autonomous safety-critical control system.

## Educational and Portfolio Focus

The implementation is intended as an educational and portfolio-grade operations analytics prototype that demonstrates applied NLP engineering patterns, not a production-certified incident investigation platform.
