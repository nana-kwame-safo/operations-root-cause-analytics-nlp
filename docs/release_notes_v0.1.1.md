# v0.1.1 — Live Prediction and UI Theme Polish

This release finalizes live demo readiness for Operations Root Cause Analytics with NLP.

## Highlights

- Live prediction is enabled on Render.
- The trained aviation model artifact is hosted externally on Hugging Face.
- Render loads the model artifact through `MODEL_ARTIFACT_URL`.
- `model.joblib` is not committed to GitHub.
- Raw data remains excluded from version control.
- Light and dark theme screenshots were added and refreshed.
- The model artifact status banner was improved in the UI.
- Responsible-use framing is retained across UI and documentation.
